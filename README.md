# AI Chatbot with OpenAI API

This is a simple AI chatbot built with Python and OpenAI API.

## Features
- LLM-based chatbot
- Prompt-based interaction
- Secure API key management using environment variables
- Error handling for API quota and connection issues

## Technologies
- Python
- OpenAI API
- python-dotenv
